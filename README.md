# 📝 Blog List App

A simple React application that displays a list of blog posts along with user profile information.

---

## 🚀 Features

* 👤 User profile section
* 📰 Blog list rendering
* 📅 Published date display
* ⚛️ Component-based architecture
* 📱 Clean UI layout

---

## 📁 Folder Structure

```id="fstructure"
src/
│
├── components/
│   ├── About/
│   ├── BlogItem/
│   ├── BlogList/
│   ├── Contact/
│   ├── Header/
│   ├── Home/
│   ├── NotFound/
│   └── UserInfo/
│
├── App.js
└── index.js
```

---

## 🧩 Components Overview

### 🔹 UserInfo

Displays:

* Profile image
* Name
* Role

---

### 🔹 BlogList

* Receives blog data as props
* Uses `.map()` to render list

---

### 🔹 BlogItem

Displays:

* Title
* Description
* Published date

---

### 🔹 Header / About / Contact / NotFound

* Used for routing/navigation pages

---

## 🛠️ Tech Stack

* React JS
* JavaScript (ES6+)
* CSS3

## ▶️ Run Locally

### 1. Clone the repo

```bash id="clone"
git clone https://github.com/Sandeep1208-dev/blogs-list.git
```

### 2. Go to project folder

```bash id="cd"
cd blogs-list
```

### 3. Install dependencies

```bash id="install"
npm install
```

### 4. Start the app

```bash id="start"
npm start
```
